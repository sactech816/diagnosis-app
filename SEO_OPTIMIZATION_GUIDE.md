# SEO最適化ガイド - 診断クイズ作成メーカー

## 実施した改善内容

### 1. タイトルタグの最適化

**変更前:**
```
診断クイズメーカー
```

**変更後:**
```
診断クイズ作成メーカー | AIで簡単作成
```

**改善ポイント:**
- 「作成」というキーワードを追加し、ユーザーの検索意図に合致
- 「AIで簡単作成」を追加し、サービスの特徴を明確化
- 検索結果でのクリック率向上を期待

### 2. メタディスクリプションの強化

**変更前:**
```
AIが自動で診断・検定・占いを作成。集客や教育に使えるクイズ作成ツール。無料で診断・性格テスト・心理テストを作成・公開できます。
```

**変更後:**
```
診断クイズ作成メーカーは、AIが自動で診断・検定・占いを作成できる無料ツール。性格診断、心理テスト、適職診断などを簡単に作成・公開。集客やマーケティング、教育に最適なクイズメーカーです。
```

**改善ポイント:**
- サービス名を冒頭に配置
- 具体的な診断タイプを列挙（性格診断、心理テスト、適職診断）
- ユーザーベネフィットを明確化

### 3. キーワードの拡充

**追加したキーワード:**
- 診断クイズ作成メーカー（メインキーワード）
- 診断メーカー
- クイズ作成
- 診断作成
- 性格診断
- 心理テスト
- 適職診断
- 占い作成
- AIクイズ
- 無料診断ツール
- 集客ツール
- マーケティングツール
- 診断コンテンツ
- クイズメーカー

### 4. 構造化データの強化（Schema.org）

**追加・改善した項目:**
- `alternateName`: 別名を追加（診断メーカー、クイズ作成ツール等）
- `featureList`: より具体的な機能リストに更新
- `keywords`: 検索キーワードを明示的に追加

これにより、Google検索のリッチリザルトやAI検索エンジンでの理解が向上します。

### 5. robots.txt の作成

**内容:**
- 検索エンジンのクロールを最適化
- AI検索エンジン（GPTBot, Claude-Web, CCBot等）を明示的に許可
- APIエンドポイントや管理画面はクロール対象外に設定

**配置場所:** `public/robots.txt`

### 6. sitemap.xml の自動生成

**内容:**
- 全ての公開ページをサイトマップに含める
- 更新頻度と優先度を適切に設定
- 検索エンジンのインデックス効率を向上

**配置場所:** `app/sitemap.ts`（Next.jsが自動的に `/sitemap.xml` として公開）

## 検索エンジン別の最適化

### Google検索
- ✅ タイトルタグに主要キーワードを含める
- ✅ メタディスクリプションを120-160文字で最適化
- ✅ 構造化データ（Schema.org）を実装
- ✅ robots.txtとsitemap.xmlを設置

### AI検索（ChatGPT, Perplexity, Claude等）
- ✅ 構造化データで機能を明確に記述
- ✅ alternateName で別名を提供
- ✅ robots.txtでAIクローラーを明示的に許可
- ✅ キーワードフィールドで検索意図を明確化
